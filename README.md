# PlushLaMode


## An app displaying dresses and accesories and allows users to customize their choices


[Visit Website](https://plushlamode.herokuapp.com)




# Technologies Used:

1.**Python**

2.**CSS**

3.**MongoDB**

4.**Authentization/Authorization**

4.**Materialize**

5.**Heroku**

6.**PostgreSQL**
